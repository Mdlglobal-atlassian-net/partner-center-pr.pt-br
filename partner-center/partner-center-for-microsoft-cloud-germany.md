# <a name="partner-center-for-microsoft-cloud-germany"></a>Partner Center do Microsoft Cloud Alemanha

**Aplicável a**

-  Partner Center do Microsoft Cloud Alemanha

## <a name="microsoft-cloud-germany-meeting-german-data-protection-requirements"></a>Microsoft Cloud Alemanha: atendendo aos requisitos alemães de proteção de dados 

o Microsoft Cloud Alemanha foi projetada para atender aos rígidos requisitos de proteção de dados da Alemanha e inclui um objeto de confiança de dados alemão que controla o acesso físico e lógico a dados de clientes de acordo com a legislação alemã. Todos os dados do cliente, e os sistemas que processam ou controlam o acesso aos dados do cliente, residem em datacenters alemães. A rede entre esses datacenters é isolada e dedicada na Alemanha.

o Microsoft Cloud Alemanha fornece uma instância separada do [Microsoft Azure](https://go.microsoft.com/fwlink/?linkid=847992), do Office 365 e do Microsoft Dynamics, entregue de datacenters na Alemanha. Consequentemente, os clientes comerciais da UE (União Europeia) e da EFTA (Associação Europeia de Livre Comércio) podem armazenar e gerenciar dados do cliente em conformidade com as leis e os regulamentos alemães aplicáveis, bem como as principais normas internacionais. Os funcionários da Microsoft não podem acessar dados do cliente sem a aprovação e a supervisão do cliente ou do objeto de confiança de dados alemão.

Consulte a [Folha de dados do Microsoft Cloud Alemanha](http://download.microsoft.com/download/6/1/3/613C9ECB-9167-4EF5-B131-3BAD8D8A126C/Microsoft_Cloud_Germany_Datasheet.pdf) para obter mais informações.

## <a name="partner-center-for-microsoft-cloud-germany-selling-german-cloud-solutions-in-eu-and-efta"></a>Partner Center do Microsoft Cloud Alemanha: vendendo soluções na nuvem alemães na UE e na EFTA

Depois de se registrar no programa CSP (Provedor de Solução na Nuvem) do Microsoft Cloud Alemanha, você usará o Partner Center para vender soluções na nuvem específicas da Alemanha e serviços para clientes em países/regiões da união Europeia e da EFTA.

## <a name="frequently-asked-questions-about-the-csp-program-for-microsoft-cloud-germany"></a>Perguntas frequentes sobre o programa CSP para Microsoft Cloud Alemanha

Use estas perguntas frequentes para saber mais sobre o programa Provedor de Soluções na Nuvem (CSP) para a Microsoft Cloud Alemanha. Você também pode entrar em contato com o seu gerente de conta para obter informações adicionais. 

**O que é Microsoft Cloud Alemanha?** 

A Microsoft Cloud Alemanha combina serviços líderes de mercado de nuvem da Microsoft com uma exclusiva arquitetura de gerenciamento de sistema e de rede dedicada localizados na Alemanha. Nesse modelo exclusivo, o Data Trustee, uma empresa alemã independente, controla o acesso aos dados do cliente e não o compartilha com terceiros, exceto conforme indicado pelo cliente ou exigido pelas leis alemãs. Isso também é conhecido como Soberania de Dados. 

**Quem está qualificado para o programa CSP da Microsoft Cloud Alemanha?**

Parceiros da União Europeia (UE), da Associação Europeia de Livre Comércio (EFTA) e do Reino Unido (UK) são qualificados para o programa CSP da Microsoft Cloud Alemanha e podem comercializar a clientes na União Europeia, EFTA e Reino Unido. 

**CSP Comercial é um programa de parceria existente. Em que difere o programa CSP da Microsoft Cloud Alemanha?**

O programa CSP da Microsoft Cloud Alemanha é diferente do CSP Comercial nas seguintes áreas: 
-   Qualificação: o programa CSP da Microsoft Cloud Alemanha está disponível apenas para parceiros da União Europeia, da EFTA e do Reino Unido. 
-   Adesão: os parceiros precisam aderir ao programa CSP da Microsoft Cloud Alemanha separadamente do CSP Comercial. 
-   Licenciamento: os documentos de licenciamento para o programa CSP da Microsoft Cloud Alemanha estão atualizados para refletir o papel do Data Trustee. Informações adicionais sobre as alterações de licenciamento aparecem abaixo. 
-   Funcionalidade do Partner Center: determinados recursos do Partner Center não estarão habilitados para o programa CSP da Microsoft Cloud Alemanha para atender aos requisitos de soberania de dados. Os parceiros podem usar o Portal Online da Microsoft (MOP) ou Azure Active Directory (Azure AD) ou API Gráfica. 

**Em que difere o licenciamento para o programa CSP para a Microsoft Cloud Alemanha do licenciamento para CSP Comercial?**

O Contrato de Revendedor de Nuvem da Microsoft (MCRA) e o Acordo de Distribuidor Microsoft Cloud (MCDA) para o programa CSP da Microsoft Cloud Alemanha diferem do MCRA Comercial e do MCDA Comercial nas duas maneiras a seguir: 
-   Eles exigem que os clientes recebam o Acordo do Microsoft loud (MCA) para Cloud Alemanha com o aditivo de Termos de Serviço Online para Cloud Alemanha 
-   Eles incluem modificações feitas MCA de requisitos de dados do cliente para a Cloud Alemanha difere do MCA Comercial destas duas maneiras: 
-   Ele inclui o aditivo para Termos de Serviços Online, que inclui os compromissos de proteção de dados adicionais pela Microsoft para hospedagem de dados, o papel do Data Trustee, e o acesso a dados do cliente 
-   Ele inclui o Acordo do Data Trustee. O Anexo 1 do aditivo contém um Acordo Cliente-Data Trustee, pelo qual a T-Systems se compromete diretamente no que se refere a controle e acesso a dados do cliente 

**Quais são as qualificações para comercializar o Programa CSP da Microsoft Cloud Alemanha?**

Os critérios de qualificação para comercializar o Programa CSP da Microsoft Cloud Alemanha equivalem aos critérios de qualificação para CSP Comercial, disponíveis para Parceiros Diretos e Revendedores Indiretos. Você não precisa ser novamente indicado para ser um Revendedor Indireto no Programa CSP da Microsoft Cloud Alemanha se você for um Revendedor Indireto aprovado no Programa CSP Comercial. 

**Sou atualmente um parceiro. O que é necessário fazer para comercializar o Programa CSP da Microsoft Cloud Alemanha?**

Os parceiros precisam aderir ao Programa CSP da Microsoft Cloud Alemanha. (A adesão ao Microsoft Cloud Alemanha é separada da adesão ao CSP Comercial). Uma vez tendo aderido, você pode prospectar clientes e ofertar o Microsoft Cloud Alemanha. Se você está aproveitando APIs, obtenha mais informações sobre APIs. 

**Ainda não sou um parceiro CSP. Onde encontro mais informações sobre como me tornar um parceiro CSP?**

Visite portal MPN para saber mais sobre modelos de registro disponíveis para parceiros. 

**Os preços para o cliente do Programa CSP da Microsoft Cloud Alemanha são os mesmos que o do CSP Comercial?**

Não. Os preços refletirão os custos adicionais para prover o valor exclusivo da Microsoft Cloud Alemanha. 

**O modelo de descontos do parceiro do Programa CSP da Microsoft Cloud Alemanha é o mesmo que o do CSP Comercial?**

Sim. O modelo de descontos do parceiro do Programa CSP da Microsoft Cloud Alemanha é idêntico ao do CSP Comercial. 

**Existe alguma diferença nos incentivos para parceiro do Programa CSP da Microsoft Cloud Alemanha daqueles do CSP Comercial?**

Não. Os incentivos para parceiro do Programa CSP da Microsoft Cloud Alemanha são os mesmos do CSP Comercial. Para saber mais sobre incentivos para parceiros, visite o portal MPN com as informações mais recentes. 

**Há alguma diferença na cobrança entre o Programa CSP da Microsoft Cloud Alemanha e o do CSP Comercial?**
Não. Os critérios de cobrança e os requisitos para o programa CSP da Microsoft Cloud Alemanha são iguais aos do CSP Comercial. Para saber mais sobre cobrança, analise a seção Recursos no portal MPN. 

**Os requisitos de suporte para o Programa CSP da Microsoft Cloud Alemanha são os mesmos que os do CSP Comercial?**

Sim. Para saber mais sobre suporte, revise a seção Recursos no portal MPN. 

**Para quais segmentos de mercado posso comercializar?**

Uma vez aprovado como parceiro do CSP MCD, você pode comercializar para qualquer cliente comercial da União Europeia, da EFTA e do Reino Unido. 

**Quais produtos/serviços estarão disponíveis no Programa CSP da Microsoft Cloud Alemanha?**

No lançamento, o Microsoft Office 365 Alemanha e o Microsoft Azure Alemanha estarão disponíveis. O Microsoft Dynamics 365 Alemanha (CRMOL) está previsto para o segundo trimestre de 2017. Vamos adicionar outros serviços quando estes forem disponibilizados. 

**O programa CSP da Microsoft Cloud Alemanha oferece todos os serviços do Azure?**

Infraestrutura como Serviço (IaaS) e plataforma fundamental e serviços de dados estão agora disponíveis. Esta previsão é atualizada periodicamente e vamos prover documentação tal como documentação do CSP Comercial do Partner Center (autenticação obrigatória).

**É possível passar para o Microsoft Cloud Alemanha dados hospedados nos datacenter da Microsoft em Dublin ou Amsterdã?**

Para facilitar e manter a integridade do Microsoft Cloud Alemanha, não há nenhum mecanismo para transferir do atual ambiente Microsoft Cloud do cliente para o Microsoft Cloud Alemanha. Se um cliente/parceiro tem um ambiente online Office 365, Azure ou Dynamics CRM, ele ficará responsáveis por transferir dados para o novo ambiente na Microsoft Cloud Alemanha.
